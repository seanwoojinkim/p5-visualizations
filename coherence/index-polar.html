Coherence Visualization - Polar H10 Edition

How to Use

  • P - Toggle Polar H10 mode (real HRV data)
  • X - Start/Stop session recording (Polar H10 mode only)
  • S - Toggle simulation mode
  • B - Toggle breathing guide
  • ↑ ↓ - Adjust breathing rate
  • C - Toggle control panel
  • Space - Pause/unpause
  • R - Reset simulation
  • ← → - Adjust coherence (manual mode)

Using Polar H10

  1. Start the HRV Monitor service:
    cd /workspace/hrv-monitor && ./run.sh
  2. Wear your Polar H10 with moistened electrodes
  3. Wait for the service to connect (~10 seconds)
  4. Press P in this visualization
Note: The HRV Monitor service must be running on your Mac (not in Docker) for Bluetooth access.

Modes

  • Manual: Control coherence with slider/arrow keys
  • Simulation: Automated biometric simulation
  • Polar H10: Real-time heart rate coherence
