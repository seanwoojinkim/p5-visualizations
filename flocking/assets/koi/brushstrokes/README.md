# Brush Texture Templates

These PNG files are templates for authentic sumi-e (Japanese ink painting) brush textures used to render koi fish spots and body parts.

## Files

### spot.png (256×256)
**Purpose**: Ink dab texture for koi spots (replacing procedural ellipses)
**Specifications**:
- Circular/organic ink spot shape
- Transparent background (PNG with alpha channel)
- Black ink with varying opacity for authentic brush effect
- Soft, feathered edges
- Current: Simple circular placeholder

### body.png (512×256)
**Purpose**: Horizontal flowing brush stroke for body texture
**Specifications**:
- Horizontal brush flow orientation
- Transparent background
- Fiber detail and ink variation
- Current: Simple ellipse placeholder

### fin.png (256×128)
**Purpose**: Delicate, wispy stroke for fin textures
**Specifications**:
- Light, delicate appearance
- Transparent background
- Wispy, flowing character
- Current: Simple ellipse placeholder

### tail.png (512×128)
**Purpose**: Flowing, dynamic stroke for tail texture
**Specifications**:
- Flowing, dynamic appearance
- Transparent background
- Directional brush flow
- Current: Simple ellipse placeholder

### paper.png (1024×1024)
**Purpose**: Paper grain texture for background (optional)
**Specifications**:
- Fine paper grain texture
- Transparent background with subtle texture
- Tileable for large surfaces
- Current: Simple semi-transparent rectangle placeholder

## Usage

Replace these placeholder files with your own brush texture images. The code will automatically load and use them.

**Important**: Maintain the exact filenames and dimensions for best results.

## Creating Authentic Brush Textures

Tips for creating realistic sumi-e brush textures:
- Use actual brush scans or digital painting tools
- Vary opacity to create organic, natural appearance
- Include fiber detail and ink bleeding effects
- Keep transparent backgrounds (PNG format required)
- Black ink on transparent background (color will be applied via tint)
