# Lilypad Assets

This directory should contain PNG images of lily pads for the pond visualization.

## Required Files

- `lilypad-1.png` - First lilypad variation
- `lilypad-2.png` - Second lilypad variation
- `lilypad-3.png` - Third lilypad variation

## Specifications

- **Format**: PNG with transparency
- **Recommended size**: 200x200px to 400x400px
- **Style**: Should match the watercolor/sumi-e aesthetic
- **Features**:
  - Round or slightly irregular oval shapes
  - V-notch cut at one edge (characteristic of lily pads)
  - Green tones (#4a7c59, #5a8c69, #4d7f5c or similar)
  - Visible veins radiating from center
  - Semi-transparent for layering effect
  - Subtle variations between the three images

## Tips

- Use semi-transparent backgrounds for natural blending with water
- Add darker patches or rings for aged/mature appearance
- Include subtle texture for organic feel
- Consider varying sizes slightly (lilypad-3 could be larger/more mature)
